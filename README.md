🎮 Tic Tac Toe (Python)

A simple command-line Tic Tac Toe game built using Python.
Two players can play the game in the terminal by entering the board positions.

📌 Features

Classic 3×3 Tic Tac Toe board

Two-player gameplay (X vs O)

Input validation

Detects:

Player win 🏆

Game tie 🤝

Clean and readable terminal board display

🖥 Board Layout

Players enter positions according to this grid:

     1     2     3
   ----------------
 1 | 11 | 12 | 13 |
   ----------------
 2 | 21 | 22 | 23 |
   ----------------
 3 | 31 | 32 | 33 |
   ----------------

Example input:

Enter position: 22

This places your symbol in the center of the board.

▶️ How to Run
1. Clone the repository
git clone https://github.com/your-username/tic-tac-toe.git
2. Go to the project folder
cd tic-tac-toe
3. Run the game
python tictactoe.py
🎲 Game Rules

Player X and Player O take turns.

Enter a valid position from the available list.

The first player to align 3 symbols in a row, column, or diagonal wins.

If all spaces are filled and nobody wins → Tie.
